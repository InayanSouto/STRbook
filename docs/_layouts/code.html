STRbook

Supplementary package STRbook is available here.

To install STRbook in R, first install devtools and then run:

library(devtools)
install_github("andrewzm/STRbook")

Run the R Labs in your browser!

You now can run the R Labs in your browser by simple clicking here. add This link will load RStudio in your browser; you will find all the Labs in the "scripts" folder (look at the bottom-right hand side of the RStudio pane). Simply double click on a script to open it, and go through it line by line (press Ctrl+Enter to do this quickly). Just pressing the "Source" button to run the Lab at once doesn't always work. Note that a couple of Labs which run FRK have been changed slightly to satisfy memory limits on the online platform (a warning is issued so you know where we did the change). Note also that the online platform may take a while to load the first time you click on it.
